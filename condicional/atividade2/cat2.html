Atividade - Número Inteiro


Usando a estrutura condicional "Se(If())", quando o usuário apertar o botão "Enviar"
com o campo vazio, o site vai informar que o campo deve ser preenchido, assim que o
campo for preenchido, o site verifica o número digitado e diz se o numero é PAR ou ÍMPAR.
